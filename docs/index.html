1. 用語の定義

本規約およびプライバシーポリシーにおいて、以下の用語は次の意味を有するものとします：

  • 「本アプリ」とは、当方が提供するメモ帳アプリを指します。
  • 「ユーザー」とは、本アプリをダウンロードし使用する者を指します。
  • 「データ」とは、本アプリを通じてユーザーが作成、保存するすべての情報を指します。

2. アプリの基本機能

本アプリは、以下の基本機能のみを提供します：

  • メモの作成
  • メモの閲覧
  • メモの編集
  • メモの削除
  • メモの自動保存

3. データの保存と管理

すべてのデータは端末内にのみ保存され、外部に送信されることはありません。データは以下の場合に削除される可能性があります：

  • アプリのアンインストール
  • 端末の設定画面からのアプリデータの削除
  • アプリのストレージの消去
  • 端末の初期化
  • OSのアップデートや端末の仕様変更
  • その他、予期せぬシステムの動作
